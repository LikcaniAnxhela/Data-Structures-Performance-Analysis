# Java Data Structures Project

## Overview

This project was developed for the Data Structures course and implements several fundamental data structures in Java. The project evaluates their behavior and performance using a dataset of 10,000 words.

## Implemented Data Structures

- Singly Linked List
- Doubly Linked List
- Doubly Linked List (Header without Tail)
- Doubly Linked List (Tail without Header)
- Binary Search Tree (BST)
- AVL Tree
- Hash Table with Linear Probing

## Features

- File processing and word loading
- Union and Intersection operations
- Tree insertion, searching, and deletion
- AVL balancing operations
- Hash table collision handling
- Runtime performance comparison

## Technologies

- Java
- Maven
- NetBeans IDE

## Learning Outcomes

- Data Structures implementation
- Algorithm analysis
- Performance measurement
- File processing
- Object-Oriented Programming
